Misclick Hypothesis

A boolean flag indicating if the user pushed a button external to the task.

    H0: There is no significant difference between Verson A and Verson B's misclick rate.

    H1: Version B will have a lower misclick rate than Version A.

  • In Version A, the colors of all the buttons are the same and there is no divider betwween the slots so it is difficult to tell the difference between all the appointment availabilities. As a result, I predict there will be a higher rate of misclicks. In my modified version, there is a clear distinciton between not only the different appointment slots but also with the buttons. In version B, I have divided the appointments with a horizontal line and made the button I want the user to click on a pink color that is very distinct from the rest of the gray colored ones. That is why I predict Version B will have a lower misclick rate due to the color distincitons. With this information in mind, I believe that we will end up rejecting the null hypothesis.

Time Spent On Page Hypothesis

Total time (in milliseconds) that the user spent on the page.

    H0: There is no significant difference in the amount of time a user spends viewing Version A compared to Version B.

    H1: Users will spend a greater amount of time viewing Version A compared to Version B.

  • As mentioned above, there is no clear distinciton between the various appointment options in Version A. In Version B, a user can scan through the divided option slots and use color heiarchy to help them quickly distinguish which appointment they intend to chose. That is why I believe that Version A users will spend more time looking at the screen than they do with Version B. With this information in mind, I believe that we will end up rejecting the null hypothesis.
